2. Quais dados coletamos

Coletamos diferentes dados dependendo de quem usa a plataforma:

De todos os usuários (cadastro)

  • Nome, e-mail, telefone
  • Credenciais de acesso (senha, armazenada de forma criptografada — nunca em texto puro)

De pacientes

  • Dados demográficos: data de nascimento, gênero, estado civil, profissão, escolaridade
  • Dados sensíveis de saúde (categoria com proteção reforçada pela LGPD, Art. 11): queixa principal, histórico de tratamentos anteriores, diagnósticos, uso de medicação, histórico familiar, e conteúdo das sessões registrado pelo psicólogo responsável
  • Check-ins voluntários de bem-estar (humor, ansiedade, sono) que você mesmo registra

De psicólogos

  • Dados profissionais: CRP, formação, abordagem terapêutica, especialidades, valores de sessão
  • Foto de perfil (opcional)

4. Quem tem acesso aos seus dados

Seguimos o princípio de menor privilégio possível: cada pessoa só acessa o que precisa para sua função.

  • O paciente acessa e edita seus próprios dados cadastrais.
  • O psicólogo responsável acessa o prontuário clínico apenas dos pacientes vinculados a ele. O conteúdo do prontuário é sigiloso — nenhum outro psicólogo ou paciente tem acesso a ele.
  • A administração da plataforma tem acesso técnico aos dados para fins de suporte, segurança e gestão de contas.
  • Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de publicidade.

Fornecedores de infraestrutura

Utilizamos o Supabase como provedor de banco de dados e autenticação, que atua como operador de dados nos termos da LGPD (Art. 5º, VII). O acesso ao banco é protegido por políticas de segurança em nível de linha (Row Level Security), que impedem que um usuário leia dados de outro mesmo em caso de falha na interface do site.

5. Por quanto tempo guardamos seus dados

Mantemos os dados de prontuário pelo prazo exigido pelas normas do Conselho Federal de Psicologia e pela legislação aplicável a registros de saúde. Dados de conta podem ser mantidos enquanto sua conta estiver ativa e, após o encerramento, pelo prazo necessário para cumprimento de obrigação legal.

6. Como protegemos seus dados

  • Conexão criptografada (HTTPS) em toda a plataforma
  • Senhas armazenadas com criptografia, nunca em texto puro
  • Controle de acesso por linha no banco de dados (RLS), garantindo que cada pessoa só veja o que tem permissão de ver
  • Segregação entre dados clínicos (sigilo do psicólogo) e dados de gestão da conta
  • Proteção contra inserção de código malicioso em campos de texto (XSS)

7. Seus direitos como titular dos dados

Conforme o Art. 18 da LGPD, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários
  • Solicitar a portabilidade dos dados a outro fornecedor
  • Eliminar dados tratados com seu consentimento (respeitadas as obrigações legais de retenção de prontuário)
  • Revogar o consentimento, quando aplicável
  • Solicitar informação sobre com quem seus dados são compartilhados

8. Menores de idade

O atendimento de menores de idade na plataforma requer consentimento e supervisão de responsável legal, conforme as normas éticas do Conselho Federal de Psicologia.
